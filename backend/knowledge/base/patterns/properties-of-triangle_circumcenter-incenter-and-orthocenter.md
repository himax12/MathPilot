### Example 1 (Year: 2005)

**Question:**

In a triangle $$ABC$$, let $$\angle C = {\pi  \over 2}$$. If $$r$$ is the inradius and $$R$$ is the circumradius of the triangle $$ABC$$, then $$2(r+R)$$ equals :

**Solution:**

We know by sin c rule 
<br><br>$${c \over {\sin C}} = 2R \Rightarrow c = 2R\sin C$$ 
<br><br>$$ \Rightarrow c = 2R$$
<br><br>$$\left( \, \right.$$as $$\,\,\,\angle C = {90^ \circ }$$ $$\left. \, \right)$$
<br><br>Also $$\tan {C \over 2} = {r \over {s - c}}$$ 
<br><br>$$ \Rightarrow \tan {\pi  \over 4} = {r \over {s - c}}$$
<br><br>$$\left( \, \right.$$as $$\,\,\,\angle C = {90^ \circ }$$ $$\left. \, \right)$$
<br><br>$$ \Rightarrow r = s - c = {{a + b - c} \over 2}$$
<br><br>$$ \Rightarrow 2r + c = a + b$$
<br><br>$$ \Rightarrow 2r + 2R = a + b$$ 
<br><br>(using $$c=2R$$)

### Example 3 (Year: 2022)

**Question:**

<p>Let a, b and c be the length of sides of a triangle ABC such that $${{a + b} \over 7} = {{b + c} \over 8} = {{c + a} \over 9}$$. If r and R are the radius of incircle and radius of circumcircle of the triangle ABC, respectively, then the value of $${R \over r}$$ is equal to :</p>

**Solution:**

<p>$${{a + b} \over 7} = {{b + c} \over 8} = {{c + a} \over 9} = \lambda $$</p>
<p>$$a + b = 7\lambda $$</p>
<p>$$b + c = 8\lambda $$</p>
<p>$$c + a = 9\lambda $$</p>
<p>$$a + b + c = 12\lambda $$</p>
<p>$$\therefore$$ $$a = 4\lambda ,\,b = 3\lambda ,\,c = 5\lambda $$</p>
<p>$$S = {{4\lambda  + 3\lambda  + 5\lambda } \over 2} = 6\lambda $$</p>
<p>$$\Delta  = \sqrt {S(s - a)(s - b)(s - c)}  = \sqrt {(6\lambda )(2\lambda )(3\lambda )(\lambda )}  = 6{\lambda ^2}$$</p>
<p>$$R = {{abc} \over {4\Delta }} = {{(4\lambda )(3\lambda )(5\lambda )} \over {4(6{\lambda ^2})}} = {5 \over 2}\lambda $$</p>
<p>$$r = {\Delta  \over s} = {{6{\lambda ^2}} \over {6\lambda }} = \lambda $$</p>
<p>$${R \over r} = {{{5 \over 2}\lambda } \over \lambda } = {5 \over 2}$$</p>

### Example 5 (Year: 2024)

**Question:**

<p>Let $$\left(5, \frac{a}{4}\right)$$ be the circumcenter of a triangle with vertices $$\mathrm{A}(a,-2), \mathrm{B}(a, 6)$$ and $$C\left(\frac{a}{4},-2\right)$$. Let $$\alpha$$ denote the circumradius, $$\beta$$ denote the area and $$\gamma$$ denote the perimeter of the triangle. Then $$\alpha+\beta+\gamma$$ is</p>

**Solution:**

<p>$$\begin{aligned}
& A(a,-2), B(a, 6), C\left(\frac{a}{4},-2\right), O\left(5, \frac{a}{4}\right) \\
& A O=B O \\
& (a-5)^2+\left(\frac{a}{4}+2\right)^2=(a-5)^2+\left(\frac{a}{4}-6\right)^2 \\
& a=8 \\
& A B=8, A C=6, B C=10 \\
& \alpha=5, \beta=24, \gamma=24
\end{aligned}$$</p>
