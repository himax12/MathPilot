# Hyperbola: Question Based On Basic Definition And Parametric Representation

**JEE Frequency**: 10 years | **Questions**: 42

## Hyperbola: Basic Definition and Parametric Representation - JEE Knowledge Base

**JEE Frequency:** High (10 years, 42 questions)

**1. Pattern Recognition:**

*   Problems directly use the definition of a hyperbola (difference of distances to foci is constant).
*   Parametric coordinates are frequently used to simplify equations, especially when dealing with tangents/normals.
*   Questions involve relating hyperbola parameters (a, b, e) to other geometric properties like foci, directrix, latus rectum.

**2. Core Formulas:**

*   Equation of Hyperbola: $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$
*   Eccentricity: $e = \sqrt{1 + \frac{b^2}{a^2}}$
*   Foci: $(\pm ae, 0)$
*   Directrices: $x = \pm \frac{a}{e}$
*   Parametric Coordinates: $(a\sec\theta, b\tan\theta)$

**3. Standard Approach:**

1.  **Identify Given Information:** Clearly write down the given parameters (a, b, e, foci, directrix, etc.).
2.  **Relate to Definition:** If the problem involves distances to foci, use the definition: $|PS_1 - PS_2| = 2a$.
3.  **Utilize Eccentricity:**  Use $e = \sqrt{1 + \frac{b^2}{a^2}}$ to relate 'a' and 'b' if not directly given.
4.  **Apply Parametric Form:** If the question deals with points on the hyperbola or involves tangents/normals, consider using the parametric form $(a\sec\theta, b\tan\theta)$.
5.  **Solve and Substitute:** Manipulate the equations obtained and substitute the values to find the required parameters.

**4. Quick Tips:**

*   **Focus on Foci/Directrix Relationship:** $ae$ and $a/e$ are key. Remember their connection to the geometry.
*   **Pythagorean Relation (modified):** Remember $b^2 = a^2(e^2 - 1)$ and use it to quickly relate 'a', 'b', and 'e'.

**5. Common Mistakes:**

*   **Confusing Hyperbola and Ellipse Formulas:** Carefully distinguish between the formulas for eccentricity, foci, and directrices.
*   **Sign Errors:** Pay close attention to signs, especially when calculating distances and using the definition.
*   **Ignoring Parametric Constraints:** When using parametric coordinates, ensure that the solution satisfies the equation of the hyperbola.
