**Solution:**

<br><br>$$\sin \left( {\alpha  - \beta } \right) = {5 \over {13}} \Rightarrow \tan \left( {\alpha  - \beta } \right) = {5 \over {12}}$$

### Example 4 (Year: 2019)

**Question:**

If cos($$\alpha $$ + $$\beta $$) = 3/5 ,sin ( $$\alpha $$ - $$\beta $$) = 5/13 and
0 &lt; $$\alpha , \beta$$ &lt; $$\pi \over 4$$, then tan(2$$\alpha $$) is equal to :

**Solution:**

Given $$0 &lt; \alpha  &lt; {\pi  \over 4}$$
<br><br>and $$0 &lt; \beta  &lt; {\pi  \over 4}$$
<br><br>$$ \therefore $$ $$0 &gt;  - \beta  &gt;  - {\pi  \over 4}$$
<br><br>$$ \therefore $$ $$0 &lt; \alpha  + \beta  &lt; {\pi  \over 2}$$
<br><br>and $$ - {\pi  \over 4} &lt; \alpha  - \beta  &lt; {\pi  \over 4}$$
<br><br>As cos($$\alpha $$ + $$\beta $$) = 3/5
<br><br>so $${\tan \left( {\alpha  + \beta } \right) = {4 \over 3}}$$
<br><br>As sin( $$\alpha $$ - $$\beta $$) = 5/13
<br><br>so $${\tan \left( {\alpha  - \beta } \right) = {5 \over {12}}}$$
<br><br>Now tan(2$$\alpha $$) = tan($$\alpha $$ + $$\beta $$ + $$\alpha $$ - $$\beta $$)
<br><br>= $${{\tan \left( {\alpha  + \beta } \right) + \tan \left( {\alpha  - \beta } \right)} \over {1 - \tan \left( {\alpha  + \beta } \right)\tan \left( {\alpha  - \beta } \right)}}$$
<br><br>= $${{{4 \over 3} + {5 \over {12}}} \over {1 - {4 \over 3} \times {5 \over {12}}}}$$ = $${{63} \over {16}}$$
