### Example 1 (Year: 2019)

**Question:**

If&nbsp;&nbsp;0 $$ \le $$ x &lt; $${\pi  \over 2}$$, &nbsp;then the number of values of x for which sin x $$-$$ sin 2x + sin 3x = 0, is :

**Solution:**

sin x $$-$$ sin 2x + sin 3x = 0&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$$x \in \left[ {0,{\pi  \over 2}} \right)$$
<br><br>$$ \Rightarrow $$&nbsp;&nbsp;(sin3x + sinx) $$-$$ sin2x = 0
<br><br>$$ \Rightarrow $$&nbsp;&nbsp;2sin2x.cos2x $$-$$ sin2x = 0
<br><br>$$ \Rightarrow $$&nbsp;&nbsp;sin2x (2cosx $$-$$ 1) = 0
<br><br>sin 2x = 0
<br><br>x = 0
<br><br>and cos x = $${1 \over 2}$$
<br><br>and x = $${\pi  \over 3}$$
<br><br>two solutions

### Example 2 (Year: 2021)

**Question:**

If $$\sqrt 3 ({\cos ^2}x) = (\sqrt 3  - 1)\cos x + 1$$, the number of solutions of the given equation when $$x \in \left[ {0,{\pi  \over 2}} \right]$$ is __________.

**Solution:**

$$\sqrt 3 ({\cos ^2}x) = (\sqrt 3  - 1)\cos x + 1$$
<br><br>$$ \Rightarrow $$ $$\sqrt 3 {\cos ^2}x - \sqrt 3 \cos x + \cos x - 1 = 0$$<br><br>$$ \Rightarrow \sqrt 3 \cos x(\cos x - 1) + (\cos x - 1) = 0$$<br><br> $$ \Rightarrow (\cos x - 1)(\sqrt 3 \cos x + 1) = 0$$<br><br>$$\cos x = 1$$<br><br>$$ \Rightarrow x = 0$$ $$ [as x \in \left[ {0,{\pi  \over 2}} \right]$$]<br><br>and $$\cos x =  - {1 \over {\sqrt 3 }}$$ (not possible in $$x \in \left[ {0,{\pi  \over 2}} \right]$$]
<br><br>$$ \therefore $$ Number of solution = 1
